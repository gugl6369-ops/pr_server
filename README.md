## 🚀 Почему эта архитектура — королевская?

| Компонент | Роль в королевстве |
|-----------|-------------------|
| **Application.php** | 👑 **Монарх** — управляет жизненным циклом приложения. Ни один запрос не проходит мимо трона. |
| **Route.php** | 🧭 **Глашатай** — объявляет, какой рыцарь (контроллер) отвечает за какой путь. |
| **Settings.php** | ⚙️ **Хранитель тайн** — держит все секретные настройки в безопасности. |
| **View.php** | 🎨 **Придворный художник** — превращает данные в прекрасные интерфейсы. |
| **Контроллеры (app/)** | ⚔️ **Рыцари** — защищают логику, обрабатывают запросы, отдают приказы художнику. |
| **routes/web.php** | 📍 **Королевский атлас** — карта, где каждый URL ведёт к своему герою. |
| **views/layouts/** | 🏛️ **Королевский дворец** — единый стиль для всех страниц. |
| **bootstrap.php** | 🔥 **Ритуал посвящения** — загружает всё королевство одним взмахом. |

## ⚡ Запуск королевства

Возведите этот сервер на трон вашей локальной машины:

```bash
# Войдите в земли королевства
cd pr_server

# Зажгите королевский огонь (запустите сервер)
php -S localhost:8000 -t public

---

## 🛠️ Установка (Возведение на трон)

Чтобы возвести этот проект на ваш локальный престол, выполните ритуал:

```bash
# Клонируйте священный репозиторий
git clone https://github.com/ваш-аккаунт/php-kingdom.git

# Войдите в палату советов (директорию проекта)
cd php-kingdom

# Запустите королевский сервер (если используете встроенный)
php -S localhost:8000 -t public
```
